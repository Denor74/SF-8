8.5. null, undefined

null

null — специальный тип в javascript. Переменная типа null может иметь лишь одно значение - null.
Буквально, null — это "ничто".

undefined

undefined — свойство глобального объекта, присваиваемое неопределенным значениям.
Буквально, undefined — это "не определён".
